php 서버 기능

1. PHP Web Service 구성도(Page)


[ 상품 리스트 ] (shop_list.php)
         │
         ├───► [ 로그인 페이지 ] (login.php) ──► (본사 ddd_user DB 검증 완료 후 세션 유지)
         │
         ├───► [ 회원가입 버튼 ] ──► [ 외부 이동 ] ──► [ 본사 회원가입 페이지 ]
         │
         └───► [ 상품 상세 페이지 ] (shop_detail.php)
                       │
                       └───► [ 즉시 구매 ] 버튼 클릭 (Form Submit)
                                     │
                                     ▼ (백엔드 처리)
                         [ 주문 처리 로직 ] (order_process.php) 
                           - PG사 연동 없이 내부 DB(주문 테이블)에 즉시 INSERT
                           - 배송지는 본사 ddd_user 테이블의 회원 정보 활용
                                     │
                                     ▼ (화면 이동)
                         [ 결제 완료 페이지 ] (order_success.php)


각 페이지별 기능 정의

① 상품 리스트 페이지 (shop_list.php)

- 기능 : 사료, 배변패드, 케이지, 하네스, 목줄 등 카테고리별 강아지 용품 목록 출력
- UI 요소 : 상품 이미지, 상품명, 가격, '상세보기' 버튼
- 비즈니스 로직 : 상품 DB에서 정보를 조회하여 루프(Loop) 문으로 화면에 렌더링

② 로그인 페이지 (login.php)

- 기능 : 서비스 이용 및 결제를 위한 사용자 인증
- 연동 로직
    - 사용자가 ID/PW 입력 ➡️ PHP 백엔드가 본사 데이터센터의 MySQL DB에 접속
    - SELECT * FROM ddd_user WHERE user_id = ? 쿼리를 수행하여 인증 처리
    - 인증 성공 시 세션($_SESSION['user_id']) 생성 후 리스트 또는 이전 페이지로 리다이렉트

③ 회원가입 요청 프로세스

- 기능 : 지사/별도 PHP 웹 서버에서 회원 데이터를 직접 쌓지 않고 본사로 일원화
- 동작 : 사용자가 회원가입 버튼을 누르면, PHP 서버가 아닌 본사 딩동독 메인 웹 사이트의 회원가입 URL(예: http://본사IP/join)로 302 리다이렉트(이동) 시킴 ➡️ 데이터 파편화 방지 및 보안성 확보

④ 상품 상세 페이지 (shop_detail.php)

- 기능 : 선택한 용품의 상세 정보(옵션 선택, 수량 설정 등) 확인
- UI 요소 : 상품 상세 설명, 수량 변경 컴포넌트, [바로 결제] 버튼
- 특이사항 : 장바구니나 별도의 '결제 내역 확인(주문서 작성) 페이지' 단계 생략

⑤ 상품 상세 페이지 (shop_detail.php)

- 사용자가 상품을 확인하고 [즉시 구매] 버튼을 누르면, 상품 번호(product_id)와 수량(quantity) 데이터가 POST 방식으로 order_process.php로 넘어갑니다.

⑥ 주문 처리 로직 (order_process.php - 화면 없음, Action만 수행)

- 실제 결제창을 띄우지 않는 대신 백엔드에서 일어나는 일
    1. 현재 로그인한 사용자의 세션($_SESSION['user_id'])이 있는지 확인합니다.
    2. 본사 ddd_user 테이블에서 해당 사용자의 이름, 전화번호, 주소 정보를 SELECT 해옵니다.
    3. 내부 주문 테이블(예: ddd_order)에 [주문번호, 유저ID, 상품ID, 수량, 배송지주소, 주문상태('결제완료')] 형태로 INSERT 쿼리를 즉시 실행합니다.
    4. 데이터 삽입이 성공하면 곧바로 order_success.php로 페이지를 넘깁니다(Redirect)
