5. 최종 PHP 페이지 목록
1) 공통
index.php : 메인 진입 페이지
db.php : MariaDB 연결
header.php : 공통 상단 메뉴
footer.php : 공통 하단 영역

2) 회원
login.php : 로그인 페이지
- 일반 보호소 회원 로그인
- 관리자 로그인 토글
login_proc.php : 로그인 처리
- Session Fixation 취약점 위치
- SQL Injection 취약점 위치
logout.php : 로그아웃 처리

3) 메인
main.php : 메인 페이지
- 랜덤 상품 8개 출력
- 상품 검색 이동
- 카테고리 이동

4) 상품 목록
shop_list.php : 상품 게시판
- 상품 목록 조회
- 카테고리 검색
- 상품명 검색
- 가격 오름차순
- 가격 내림차순
- SQL Injection 취약점 위치

5) 상품 상세
shop_detail.php : 상품 상세 조회
- 상품 정보 출력
- 수량 선택
- 장바구니 담기
- 바로 구매
cart_add.php : 장바구니 저장 처리
- CSRF 취약점 위치
order_direct.php : 즉시 구매 처리
- CSRF 취약점 위치

6) 장바구니
cart.php : 장바구니 조회
- 상품 목록 출력
- 수량 변경
- 상품 삭제
- 선택 상품 구매
cart_delete.php : 장바구니 삭제
- IDOR 취약점 위치
cart_order.php : 선택 상품 구매 처리

7) 마이페이지
mypage.php : 마이페이지
- 주문 내역 조회
- 배송 상태 조회
order_view.php : 주문 상세 조회
- 주문 정보 조회
- IDOR 취약점 위치

8) 관리자
admin_login.php : 관리자 로그인
admin_login_proc.php : 관리자 로그인 처리
product_add.php : 상품 등록
- 상품 이미지 업로드
- File Upload 취약점 위치
product_edit.php : 상품 수정
- 상품 정보 수정
- 상품 삭제
- 상품 이미지 다운로드
- File Upload 취약점 위치
admin.php : 관리자 메인 페이지
- Privilege Escalation 취약점 위치
